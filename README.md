This project is a Django-based REST API for collecting and storing resume details such as name, email, phone number, skills, experience, and uploaded resume file.

Python Version Used

Python 3.10
Django
Django REST Framework
SQLite / PostgreSQL
Git & GitHub


Installation Steps

Clone the repository:

git clone https://github.com/your-username/your-repository-name.git

pip install -r requirements.txt

python manage.py makemigrations python manage.py migrate

Example request 

email: user@example.com 
name: John Doe 
phone: 9876543210 
skills: Python, Django, REST API 
experience: 1 year backend development 
resume: resume.pdf
